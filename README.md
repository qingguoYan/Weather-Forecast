# 启动项目

npm i

npm run dev

# 获取天气 API

## 天气资源地址

https://openweathermap.org/

## 获取 API Key

注册账号，验证后，在 My API keys 找到自己的 key，复制粘贴到 api 中去，如果请求 401，请稍等几分钟，等待 api key 的激活

API只能免费免费获取未来五天(包括当天)的天气
